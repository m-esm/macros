REVIEWER NOTES — Macros

What this extension does:
Macros adds a nutrition information table overlay to Wolt Market (wolt.com) venue pages. It helps users compare grocery products by their nutritional values (calories, protein, fat, carbohydrates, fiber).

How to test:
1. Install the extension
2. Navigate to a Wolt Market venue page, e.g.:
   https://wolt.com/en/fin/helsinki/venue/wolt-market-vallila/items/meijerituotteet-munat-100
3. Wait for the page to load — a floating "Show nutrition table" button appears at the bottom-right
4. Click the button to open the nutrition table
5. Try sorting columns, using the search bar, or switching score presets

Technical details:
- The extension intercepts Wolt's own category API responses (via fetch patching in MAIN world) to capture product data as it loads
- Product nutrition info is fetched from prodinfo.wolt.com (Wolt's own nutrition service) via the background service worker (required for CORS)
- No remote code execution, no eval(), no external servers
- All data processing happens locally in the browser
- User preferences (score preset, custom weights) are stored via chrome.storage.local

Permissions justification:

storage:
- Saves the user's selected score preset (High Protein, Low Carb, Balanced, Custom)
- Saves custom scoring weights when the user adjusts the 6 nutrient sliders
- Saves the extension enabled/disabled toggle state from the popup
- No sensitive data is stored — only UI preference values
- All storage is local (chrome.storage.local), nothing is synced

Host permissions justification:
- wolt.com: Content scripts run on Wolt Market pages to inject the nutrition table UI and intercept category API responses
- prodinfo.wolt.com: The background service worker fetches product nutrition info pages (HTML containing nutrition tables) — required via service worker because content scripts cannot make cross-origin requests in MV3
- consumer-api.wolt.com: Fetches paginated category product listings to discover all items in a category beyond what the initial page load provides
